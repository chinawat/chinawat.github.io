04/18/2008 Probability Revisited: Addition Rule and Multiplication Rule
- Addition Rule: If A and B are disjoint events (the intersection is empty), then Pr(A\cup B)=Pr(A)+Pr(B).
- Multiplication Rule: If A and B are independent events, then Pr(A\cap B)=Pr(A)Pr(B). This is in fact the definition of independence of events.
Conditional Probability & Independence, Revisited
Random Variables & Independent Random Variables
> Random variable is a function from the set of possible outcomes S to any (nonempty) set V. V does not have to be a numerical set.
Expectation
> Expectation of a random variable is only defined on a numerical-valued random variable (even though random variables may give some other value than numbers--see above).
> Linearity of expectation always applies to any two numerical-valued random variables. The random variables need not be independent.
